# tbd-practice

Practice repository for Trunk-Based Development (TBD).

## What is implemented
- work directly from main
- short-lived feature branch
- fast merge back to main
- feature flag in code
- basic GitHub Actions CI pipeline

## Project files
- app.py — simple Python app with feature flag
- .github/workflows/ci.yml — basic CI workflow

## Trunk-Based Development concepts practiced
- small commits
- fast integration
- short-lived branches
- working main branch
- CI on push
- feature flags

## How to run
python3 app.py

## CI
GitHub Actions runs automatically on every push to main.

## Goal
Hands-on practice of TBD using GitHub, SSH server, console, and CI/CD.
